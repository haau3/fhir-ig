### Introduction

The haau3 FHIR Implementation Guide (HFIG) is designed to solve the interoperability problems for caregivers and patients. It is anchored on US Core 6.1.0, U.S. Core Data for Interoperability (USCDI) v3, and CARIN Consumer Directed Payer Data Exchange (Carin IG for Blue Button) as it's initial foundation.
